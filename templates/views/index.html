Перечень ликвидных инструментов

Введите тикер интересующего вас инструмента или нескольких (до 10).
Тикеры можно разделять с помощью пробела, например:
AAPL MSFT SQ PYPL.

Тикер Имя Шорт %, Лонг риск %, Шорт риск Short % of Float Short Interest Ratio (days)

На данной странице выводятся только ликвидные инструменты Тинькофф Инвестиций. Скрипт регулярно обновляет данные из первоисточника. Цель данного скрипта предоставить возможность для быстрого сравнения наличия возможности взятия в шорт и ставки риска брокера по нескольким инструментам одновременно, вместо поиска по странице.

Данные Short % of Float и Short Interest Ratio (days) берутся с сайта shortsqueeze.com при помощи моего NPM пакета shortsqueeze.

По некоторым активам можно оценить корреляцию между Шорт риском и Short % of Float. Если Шорт риск брокера и Short % of Float выше 40%, то это может быть сигналом к возможному Short Squeeze.
